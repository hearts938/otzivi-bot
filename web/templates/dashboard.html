Админ-панель

Те же функции, что и в Telegram-боте (/admin).

  • 1) Пользователи (сводка)
  • 2) Управление пользователями
  • 3) Рассылка — текст + кнопка «Старт» в бот
  • 3а) Рассылка на другое — текст + кнопка со своей ссылкой
  • 3б) Детализация по отзывам — остаток по платформам, авто в 22:00
  • 4) Задания на проверке — одобрить / отклонить отзывы
  • 5) Финансовая панель
  • 5а) Реестр выплат — номера wdN, поиск по @username или номеру
  • 5б) Статистика выплат — сегодня / неделя / месяц / всё время
  • 6) Управление балансом — начислить / списать по @username
  • 7) Тест Яндекс Карт — пул вопросов (2 случайных, только Яндекс)
  • 8) Курс звёзд
  • 9) Поддержка — ответы на обращения пользователей
  • 10) Обращение от бота — сообщение одному пользователю
  • 11–13) Платформы / кулдауны / удаление
  • 14) Заказчики и пул текстов
  • Импорт отзывов (Excel) — загрузка по шаблону
